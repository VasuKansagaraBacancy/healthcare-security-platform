Healthcare Cybersecurity Risk Management Platform
Implemented Features

1. Authentication
- User registration with Supabase Auth
- User login with Supabase Auth
- User logout
- Session-based protected routes
- Protected API access for authenticated users only

2. Roles and Access Control
- Role model implemented:
  Admin
  Security Analyst
  Viewer / Staff
- Role-aware sidebar navigation
- Role-aware mobile navigation
- Hidden unauthorized modules from navigation
- Server-side route protection for restricted pages
- Permission-based action control for UI, server actions, and APIs
- Read-only experience for viewer/staff where editing is not allowed
